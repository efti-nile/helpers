# helpers

## Pycharm
- Alt + F12 - os terminal
- Alt + F11 - py terminal
- Alt + 0 - context menu
- F5 - debug
- Alt + F5 - stop
- F6 - resume program
- F4 - toggle line breakpoint
- F3 - step over
- Alt + Left - navigate back
- Alt + Right - navigate forward
- Ctrl + PgUp - preveous tab
- Ctrl + PgDn - next tab
